# 食品科普网站

面向普通消费者的食品安全、营养标签与食品热点科普平台，帮助用户把专业食品信息转化为能理解、能判断、能行动的生活建议。

## 项目定位

本项目不做行业门户、法规数据库、企业服务或医疗建议平台。核心目标是围绕食品安全、食品营养、热点事件、食品标签和食品谣言，提供结构化、可信、易懂的科学解释。

## 核心模块

- 食品安全
- 食品营养
- 热点解读
- 标签识读
- 谣言澄清

## 第一版范围

- 首页
- 食品安全、食品营养、热点解读列表页
- 标签识读交互页
- 谣言澄清卡片页
- 文章详情页
- 搜索与分类筛选
- 权威来源说明页

## 推荐技术路线

- Next.js / React
- TypeScript
- Tailwind CSS
- Markdown / JSON 内容文件

## 第一版暂缓

第一版不加入登录、评论、收藏、会员、社区、复杂后台、完整 AI 问答和数据库系统，优先完成静态内容与轻量交互体验。

## 内容原则

- 强调风险大小、暴露剂量、适用条件、证据来源和更新时间。
- 避免制造焦虑、标题党、无来源结论、医疗化承诺和绝对化表达。
- 所有科普内容应尽量提供权威来源、关注等级和证据等级。
